Diseño de interfaz de usuario

Este proyecto se realizó desde la investigación y creación del benchmarking, SEO y copy. El diseño UI se ha creado con base a las tendencias UI 2021, se creó el UI Kit para tener un orden y uniformidad de la composición. Se respetó la linea de diseñó que tiene la empresa, con esto y las reglas de negocio, logramos realizar una segunda versión. Se ha creado con un proceso de wireframes en Miro, mockups en Figma y edición de recursos gráficos en Illustrador y Photoshop.
